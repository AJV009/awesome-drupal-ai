# Comprehensive Inventory: Drupal AI Modules on drupal.org

## Core AI Framework

### AI (Artificial Intelligence) Module
- **URL:** https://www.drupal.org/project/ai
- **Docs:** https://project.pages.drupalcode.org/ai/
- **Description:** Core AI framework with abstraction layer for 48+ providers (OpenAI, Anthropic, Mistral, Hugging Face, AWS Bedrock, Azure, Google Gemini, Google Vertex, etc.)
- **Submodules:** AI Automators, AI CKEditor, AI ECA, field widget actions
- **Relevance:** CRITICAL — the foundation of Drupal's AI ecosystem
- **Security:** SA-CONTRIB-2026-028 (info disclosure, patched)

### AI Agents Module
- **URL:** https://www.drupal.org/project/ai_agents
- **Docs:** https://project.pages.drupalcode.org/ai_agents
- **Description:** Agentic framework for creating text-to-action agents
- **Built-in Agents:** Field Type Agent, Content Type Agent, Taxonomy Agent
- **Relevance:** CRITICAL — core agentic framework

### AI Agent Handler Agent
- **URL:** https://www.drupal.org/project/ai_agent_agent
- **Description:** Meta-agent that orchestrates other agents
- **Relevance:** HIGH

### AI Drush Agents
- **URL:** https://www.drupal.org/project/ai_drush_agents
- **Description:** AI agents accessible via Drush CLI
- **Relevance:** HIGH — command-line AI interaction

### AI Migration Agent
- **URL:** https://www.drupal.org/project/ai_agent_migration

# Contributing to Awesome Drupal AI

## Adding a Tool, Skill, or Resource

1. **Check it doesn't already exist** — search the README for the tool name
2. **Open an issue** with:
   - Tool name and URL
   - Brief description (under 15 words)
   - Which category it belongs in
   - Install method (if applicable)
3. Or submit a PR directly following the format below

## Format

Each entry follows this pattern:

```markdown
- [Name](URL) — Brief description ★★★
```

With install counts where known:
```markdown
- [drupal/module_name](https://www.drupal.org/project/module_name) (1,234 installs) — Brief description ★★★
```

With star counts for GitHub repos:
```markdown
- [user/repo](https://github.com/user/repo) (★ 24) — Brief description ★★★
```

## Maturity Ratings

| Rating | Meaning |
|--------|---------|
| ★★★★★ | Production-ready, widely adopted, actively maintained |
| ★★★★ | Stable, well-documented, recommended |
| ★★★ | Working, useful but may need refinement |
| ★★ | Early stage, experimental, or niche |
| ★ | Very early, limited utility |

## Categories

Place entries in the most specific category that fits. If a tool spans categories, list it in its primary category and note the cross-applicability.

## What Belongs Here

- Publicly available tools, modules, skills, plugins, and resources
- Relevant to Drupal development or Drupal site building with AI
- Actively maintained (or historically significant)

## What Doesn't Belong

- Internal or proprietary tools
- Tools with no Drupal relevance
- Paid tools without a free tier (unless widely used in the Drupal community)
- Duplicate entries under different names
